## Dict-like string formatting from an object by named atttributes 
Originally published: 2009-08-17 12:59:16 
Last updated: 2009-08-17 12:59:16 
Author: Jacob Oscarson 
 
Formats a string using the same notation as a dict-expansion, but\nusing an object as the source for the expansion instead. Attributes\nfrom the object are used as source to the expansions.
