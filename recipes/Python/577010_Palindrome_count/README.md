###Palindrome count

Originally published: 2010-01-16 11:21:15
Last updated: 2010-01-16 11:21:15
Author: David Lambert

'abccdadba' contains 50 palindromes, not all unique.  Examples are:\n'a' at positions 0,-4, and -1.\n'aa' at (0,4),(5,-1), and (5,-1)\nThe longest is 'abdadba'\nThis code counts the palindromes in the data variable.
