## Simple tabulator  
Originally published: 2010-11-09 12:48:43  
Last updated: 2010-11-09 12:50:06  
Author: Noufal Ibrahim  
  
This is a simple script to covert a top to bottom list of items into a left to right list.\n\n     a \n     b \n     c\n     d\n     e\n     f\n     g\n     h\n     i\n     j\n     k\n     l\n     m\n\ninto\n\n     a  b  c  d  e  f  g\n     h  i  j  k  l  m\n\n\nA few command line options allow some amount of customisation.
