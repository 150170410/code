###Split Up Text File by Line Count

Originally published: 2012-02-17 18:33:09
Last updated: 2012-02-19 23:40:47
Author: Andrew Yurisich

Splits a large text file into smaller ones, based on line count. Original file is unmodified.\n\nResulting text files are stored in the same directory as the original file.\n\nUseful for breaking up text based logs or blocks of email logins into smaller parts.
