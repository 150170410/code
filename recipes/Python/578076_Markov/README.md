Originally published: 2012-03-14 17:39:17

This program is meant to act as an example of how to use ME with\ndata that needs to be obfuscated. The functionality provided via\nthe GUI demonstrates both the ability to encrypt and decrypt all\ntext that the UTF-8 encoding can handle. Explanations come later.
