## Silmultaneous linear equation solver(any order)

Originally published: 2012-04-24 12:29:30
Last updated: 2012-04-24 12:29:30
Author: Sachin Joglekar

Solves simultaneous linear equations of any order using Crammer's rule. Required input is two lists..one for coefficients, and other for constants \neg. 2x+3y=8\n    x+4y=6 will be written as\nsimul([[2,3],[1,4]],[8,6])
