## Trifocal Tensor classOriginally published: 2012-08-28 14:16:47 
Last updated: 2012-08-28 14:16:47 
Author: J W J 
 
A class that creates a trifocal tensor from either a pair of camera matrices or three corresponding point sets. \n<code>Trifocal( Pi, Pii )</code>\n<code>Trifocal( x0, x1, x2 )</code>\nIncludes methods for testing constraints, retrieving epipoles and fundamental matrices.
