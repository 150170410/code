## Generating Balanced Parenthesis  
Originally published: 2013-02-14 22:00:22  
Last updated: 2013-02-14 22:00:22  
Author: bartender   
  
The python code generates balanced parenthesis recursively.\nExample for depth=3 i.e. 3 opening brackets and 3 closing brackets,the sample output should be:\n\n((()))\n(()())\n(())()\n()(())\n()()()\n\nThe number of such output ( 5 in above case) is a Catalan Number.
