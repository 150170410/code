## Tic Tac Toe console game 
Originally published: 2013-06-18 10:32:41 
Last updated: 2013-06-18 10:52:27 
Author: Russel Walker 
 
Just a simple text based Tic Tac Toe game for human vs computer play. \n\nThe game and AI reside in the TTTGame class. AI uses brute force search for 'perfect play'.\n\nThe CLI class provides a text based console user interface.
