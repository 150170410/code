## Recursive Multimedia (audio, video) M3U Playlist Generator 
Originally published: 2013-11-24 11:40:21 
Last updated: 2014-02-08 01:03:36 
Author: Mano Bastardo 
 
Generate an m3u playlist searching recursively\nfor multimedia files (video or audio) in the given\ndirectory.\nInformation from ID3 tags will be extracted for audio\nfiles with [FFmpeg](http://en.wikipedia.org/wiki/FFmpeg) available.
