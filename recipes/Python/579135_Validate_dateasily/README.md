## Validate data easily with JSON SchemaOriginally published: 2015-12-06 21:23:47 
Last updated: 2015-12-06 21:23:48 
Author: Vasudev Ram 
 
This recipe shows how to use the jsonschema Python library, which implements the JSON Schema specification, to easily validate your Python data. It sends good output to stdout and bad output to stderr, for demo purposes.\n\njsonschema Python library: https://pypi.python.org/pypi/jsonschema\n\nJSON Schema: http://json-schema.org/\n
