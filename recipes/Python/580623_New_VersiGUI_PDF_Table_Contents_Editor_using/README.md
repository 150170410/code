###New Version: GUI PDF Table of Contents Editor using fitz / MuPDF, wxPython

Originally published: 2016-03-16 08:42:33
Last updated: 2016-06-22 11:34:20
Author: Harald Lieder

Create or modify the bookmarks list of a PDF (ToC - table of contents).\nSupports arbitrary hierarchy levels.\nDisplay PDF while editing to control bookmark targets.\nPDF meta data are maintainable.\nSave PDF under the same or a different name.
