## Recursive find replace in files using regexOriginally published: 2016-04-26 18:34:30 
Last updated: 2016-04-28 13:24:15 
Author: ccpizza  
 
Recursively find and replace text in files under a specific folder with preview of changed data in dry-run mode\n============\n\nExample Usage\n---------------\n\n**See what is going to change (dry run):**\n\n    find_replace.py --dir project/myfolder --search-regex "\\d{4}-\\d{2}-\\d{2}" --replace-regex "2012-12-12" --dryrun\n\n**Do actual replacement:**\n\n    find_replace.py --dir project/myfolder --search-regex "\\d{4}-\\d{2}-\\d{2}" --replace-regex "2012-12-12"\n\n**Do actual replacement and create backup files:**\n\n    find_replace.py --dir project/myfolder --search-regex "\\d{4}-\\d{2}-\\d{2}" --replace-regex "2012-12-12" --create-backup\n\n\n**Same action as previous command with short-hand syntax:**\n\n    find_replace.py -d project/myfolder -s "\\d{4}-\\d{2}-\\d{2}" -r "2012-12-12" -b\n\nOutput of `find_replace.py -h`:\n\n    DESCRIPTION:\n        Find and replace recursively from the given folder using regular expressions\n    \n    optional arguments:\n      -h, --help            show this help message and exit\n      --dir DIR, -d DIR     folder to search in; by default current folder\n      --search-regex SEARCH_REGEX, -s SEARCH_REGEX\n                            search regex\n      --replace-regex REPLACE_REGEX, -r REPLACE_REGEX\n                            replacement regex\n      --glob GLOB, -g GLOB  glob pattern, i.e. *.html\n      --dryrun, -dr         don't replace anything just show what is going to be\n                            done\n      --create-backup, -b   Create backup files\n    \n    USAGE:\n       find_replace.py -d [my_folder] -s <search_regex> -r <replace_regex> -g [glob_pattern]\n
