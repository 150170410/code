###Roman Numeral Converter

Originally published: 2016-12-12 08:36:31
Last updated: 2016-12-13 05:05:52
Author: Brandon Martin

Provides two functions:\nromanToNumber - Converts a string of Roman Numerals (I,V,X,L,C,D,M) into an integer. (ignores formatting, will return best guess if improperly formatted)\nnumberToRoman - Converts an integer into a string of Roman Numerals\n\n
