###chatbox megawidget for tkinter

Originally published: 2017-02-23 22:38:12
Last updated: 2017-02-23 22:47:16
Author: Miguel Martínez López

Chatbox megawidget.\n\nArguments:\n\n   - my_nick\n\n   Nick of user chatting\n     \n   - command\n\n   Callback to call when a message is sent\n\n   - logging_file\n\n      File to log all the messages\n\n   - topic\n\n      Topic for the chatbox\n\n   - maximum_lines\n\n      Max lines to show\n\n     \n   - entry_controls\n\n      Builder of controls for the entry (optional)\n\n   - tags\n\n      List of tag configurations for the Text widget\n\n\nStyling arguments:\n\n   - timestamp_template\n\n   - scrollbar_background\n\n   - scrollbar_troughcolor\n\n   - history_background\n\n   - history_font\n\n   - history_padx\n\n   - history_pady\n\n   - history_width\n\n   - entry_font\n\n   - entry_background\n\n   - entry_foreground\n\n   - label_template\n\n   - label_font\n
