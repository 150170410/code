## Simple multicolumn listbox for tkinter  
Originally published: 2017-05-02 22:25:12  
Last updated: 2017-05-02 22:27:15  
Author: Miguel Martínez López  
  
This recipe makes easy to work a treeview widget like a table.\n\nIt has several options for styling:\n\n- heading_anchor\n- heading_font\n- heading_background\n- heading_foreground\n- cell_anchor\n- cell_background\n- cell_foreground\n- cell_font\n- cell_pady\n- height\n- padding\n- adjust_heading_to_content\n- stripped_rows\n- headers\n- selection_background\n- selection_foreground\n- field_background\n\nThe "command" parameter is a callback and its called each time a row is selected.
