## frange(), a range function with float increments

Originally published: 2001-08-07 05:00:48
Last updated: 2001-08-07 05:00:48
Author: Dinu Gherman

Sadly missing in the Python standard library, this function\nallows to use ranges, just as the built-in function range(),\nbut with float arguments.\n\nAll thoretic restrictions apply, but in practice this is\nmore useful than in theory.
