###Lexing C

Originally published: 2002-08-22 16:34:35
Last updated: 2002-08-22 16:34:35
Author: andreas kupries

Lexing C sources into a list of tokens. Demonstration of the string processing facilities of Tcl. The same approach will work for essentially all keyword based languages, for example Pascal, module, Eiffel, etc.\n<p>\nList and quoting based languages (Tcl itself, Lisp, ...) can be done too, but this is not optimal.\n<p>\nAuthor: Andreas Kupries <br>\nOriginal location: http://mini.net/tcl/3906<br>
