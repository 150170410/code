## Parse a DIS PDU

Originally published: 2009-08-29 09:49:16
Last updated: 2009-08-29 09:49:16
Author: Frank Bannon

Decodes Distributed Interactive Simulation (DIS) Protocol Data Units (PDUs) from UDP packets.\nRequires UDP package (http://tcludp.sourceforge.net)\n\nEntity State (1), Fire (2), and Detonation (3) PDUs are decoded.
